# Publishing

This repository is ready to publish as:

```text
Keseras34938976/yuzubou-codex-pet
```

## 1. Create the GitHub repository

Create a new public GitHub repository named:

```text
yuzubou-codex-pet
```

Do not initialize it with a README, license, or `.gitignore`; those files already exist here.

## 2. Push

From this folder:

```bash
cd yuzubou-codex-pet
git remote add origin https://github.com/Keseras34938976/yuzubou-codex-pet.git
git push -u origin main
```

## 3. Install URL for users

After publishing, users can install ゆず坊 with:

```bash
mkdir -p ~/.codex/pets
git clone https://github.com/Keseras34938976/yuzubou-codex-pet.git ~/.codex/pets/yuzubou
```
